#  End-to-End Retail Performance and Behavioral Analytics

## Project Overview
End-to-end retail analytics project using Python, SQL Server, and Power BI to analyze sales performance, customer behavior, product profitability, and return patterns.

## Tools Used
- Python (Pandas)
- SQL Server
- Power BI
- Excel

## Key Steps
1. Data cleaning and exploratory data analysis using Python
2. Database design and business queries using SQL
3. Interactive dashboards built in Power BI

## Dashboards
- Sales Overview
- Customer Insights
- Product Performance
- Store Performance
- Return Insights

## Key Insights
- Identified top-performing products and stores
- Analyzed customer segments driving revenue
- Studied return patterns and sales trends
